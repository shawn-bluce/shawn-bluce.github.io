Pythonдёӯзҡ„pycж–Үд»¶

еҸ‘иЎЁдәҺ 2017-12-23 жӣҙж–°дәҺ 2023-07-09

жҲ‘д»¬еңЁзј–еҶҷPythonзЁӢеәҸзҡ„ж—¶еҖҷдјҡеҸ‘зҺ°еңЁжҲ‘д»¬зҡ„зӣ®еҪ•дёӯеҸҜиғҪдјҡеҮәзҺ°дёҺжәҗд»Јз ҒеҗҢеҗҚзҡ„pycж–Үд»¶з”ҹжҲҗпјҢжҜ”еҰӮжңүдёҖдёӘжәҗз Ғж–Үд»¶жҳҜhello.pyйӮЈд№ҲеҸҜиғҪдјҡз”ҹжҲҗдёҖдёӘhello.pycж–Үд»¶еҮәжқҘпјҺиҝҷдёӘpycж–Үд»¶жҳҜPythonзҡ„еӯ—иҠӮз Ғж–Үд»¶пјҢе°ұзұ»дјјдәҺJavaдёӯзҡ„hello.classдёҖж ·пјҺ

PythonиҷҪз„¶жҳҜи§ЈйҮҠжҖ§иҜӯиЁҖпјҢдҪҶиҝҳжҳҜеҸҜд»ҘжңүдёҖдёӘзј–иҜ‘зҡ„иҝҮзЁӢпјҢеҸӘдёҚиҝҮжҳҜзј–иҜ‘жҲҗеӯ—иҠӮз Ғж–Үд»¶зҪўдәҶпјҺеҰӮжһңжҲ‘д»¬зҡ„жәҗз ҒеёҰжңүеҢ…еҗ«е…ізі»пјҢжҜ”еҰӮжәҗз Ғa.pyйҮҢйқўimportдәҶжәҗз Ғж–Үд»¶b.pyпјҢйӮЈд№ҲеңЁжү§иЎҢpython a.pyзҡ„ж—¶еҖҷе°ұдјҡе°Ҷb.pyзј–иҜ‘жҲҗb.pycпјҺ

дёӢж¬ЎеҶҚиҝҗиЎҢpython a.pyзҡ„ж—¶еҖҷи§ЈйҮҠеҷЁдјҡжЈҖжҹҘb.pyдёҺb.pycзҡ„дҝ®ж”№ж—¶й—ҙпјҢеҰӮжһңдёҖиҮҙе°ұд»ЈиЎЁжәҗз ҒжІЎжңүдҝ®ж”№иҝҮпјҢйӮЈд№Ҳе°ұеҸҜд»ҘзӣҙжҺҘи°ғз”Ёb.pycжқҘжӣҙеҝ«зҡ„жү§иЎҢзЁӢеәҸпјҢеҰӮжһңж—¶й—ҙдёҚеҗҢйӮЈе°ұиҜҒжҳҺb.pyиў«дҝ®ж”№иҝҮпјҢеҲҷдјҡйҮҚж–°зј–иҜ‘b.pyпјҺ

е…¶дёӯpycж–Үд»¶зҡ„дё»иҰҒдҪңз”ЁжҳҜз”ЁжқҘеҠ еҝ«зЁӢеәҸжү§иЎҢйҖҹеәҰзҡ„пјҢиҷҪз„¶зј–иҜ‘еҮәжқҘзҡ„pycжҳҜдәҢиҝӣеҲ¶ж–Үд»¶пјҢдёҚиғҪзңӢеҲ°еҶ…йғЁзҡ„еҶ…е®№пјҢдҪҶжҳҜиҝҳжҳҜдёҚиҰҒжҠҠиҝҷз§Қж–№ејҸдҪңдёәдҝқжҠӨжәҗз Ғзҡ„ж–№жі•пјҺеӣ дёәиҝҷз§Қpycж–Үд»¶жҳҜеҸҜд»ҘиҪ»жҳ“иў«еҸҚзј–иҜ‘зҡ„пјҢжңүеҫҲеӨҡејҖжәҗеә“еҸҜд»ҘиҪ»жқҫзҡ„еҸҚзј–иҜ‘pycж–Үд»¶пјҢз”ҡиҮійғҪжңүwebзЁӢеәҸжқҘеҸҚзј–иҜ‘pycж–Үд»¶пјҢжҜ”еҰӮиҝҷдёӘtool.luе°ұеҸҜд»ҘйҖҡиҝҮдёҠдј ж–Үд»¶pycж–Үд»¶зҡ„ж–№ејҸеҸҚзј–иҜ‘пјҺ

дҪҝз”Ёuwsgiе’ҢsupervisorйғЁзҪІDjangoзЁӢеәҸ

еҸ‘иЎЁдәҺ 2017-12-23 жӣҙж–°дәҺ 2023-07-09

0X00 дҪҝз”ЁuwsgiеҗҜеҠЁDjango

йҰ–е…Ҳе®үиЈ…uwsgiпјҢpip install uwsgiе°ұеҸҜд»ҘиЈ…еҘҪпјҺз„¶еҗҺжүҫеҲ°Djangoз”ҹжҲҗзҡ„wsgi.pyж–Үд»¶пјҢиҝҷж–Үд»¶йҖҡеёёе®һеңЁдёҺйЎ№зӣ®еҗҚеҗҢеҗҚзҡ„appзӣ®еҪ•дёӢзҡ„пјҢжҜ”еҰӮжҲ‘зҡ„йЎ№зӣ®еҗҚдёәdjango_testйӮЈд№ҲиҝҷдёӘж–Үд»¶еә”иҜҘе°ұеңЁdjango_test/wsgi.pyпјҺз„¶еҗҺжү§иЎҢuwsgi --http 0.0.0.0:8080 --wsgi-file django_test/wssgi.pyе°ұеҸҜд»Ҙз”ЁuwsgiеҗҜеҠЁдҪ зҡ„DjangoйЎ№зӣ®дәҶ.
DjangoиҮӘеёҰзҡ„python manage.py runserverз”ЁдәҺи°ғиҜ•иҝҳжҳҜеҸҜд»Ҙзҡ„пјҢдёҚиҝҮеҰӮжһңз”ЁдәҺз”ҹдә§зҺҜеўғзҡ„дёҚи®әжҳҜе®үе…ЁжҖ§иҝҳжҳҜжҖ§иғҪйғҪдёҚи¶ід»Ҙж»Ўи¶із”ҹдә§зҺҜеўғзҡ„йңҖиҰҒ.

0X01 дҪҝз”Ёsupervisorз»ҙжҢҒжңҚеҠЎеңЁзәҝ

жҲ‘д»¬зҹҘйҒ“з”ұдәҺLinuxзі»з»ҹзҡ„иҝӣзЁӢз®ЎзҗҶжңәеҲ¶пјҢеҜјиҮҙеңЁbashйҮҢеҗҜеҠЁзҡ„зЁӢеәҸеҰӮжһңйҖҖеҮәbashе°ұдјҡиҮӘеҠЁиў«killжҺүпјҺжүҖд»ҘжҲ‘д»¬йңҖиҰҒдёҖдёӘиғҪи®©иҝӣзЁӢдёҖзӣҙжҙ»дёӢеҺ»зҡ„ж–№жі•пјҢе…¶дёӯnohupжҳҜжҲ‘д»¬еёёз”Ёзҡ„ж–№жі•пјҢдёҚиҝҮиҝҷз§Қж–№жі•д№ҹеҸӘжҳҜиғҪи®©иҝӣзЁӢжҙ»еңЁеҗҺеҸ°зҪўдәҶпјҢе№іж—¶и·‘дёӘи„ҡжң¬жҲ–жҳҜе…¶д»–е°ҸзЁӢеәҸиҝҳеҸҜд»ҘпјҢеҰӮжһңжҳҜйғЁзҪІдёҖдёӘжңҚеҠЎзҡ„иҜқе°ұдёҚеҗҲйҖӮдәҶпјҺиҝҷйҮҢжҲ‘д»¬йҖүз”ЁsupervisorжқҘз»ҙжҢҒжңҚеҠЎпјҺ

0X00 е®үиЈ…fabric

дҪҝз”ЁpipеҸҜд»ҘиҪ»жқҫең°е®үиЈ…fabric

0X01 еҲқж¬Ўи°ғз”Ё

еңЁеҪ“еүҚзӣ®еҪ•дёӢеҲӣе»әдёҖдёӘеҗҚдёәfabfile.pyзҡ„ж–Үд»¶пјҢеЎ«еҶҷж–Үд»¶еҶ…е®№еҰӮдёӢпјҡ

з„¶еҗҺеңЁеҪ“еүҚзӣ®еҪ•дёӢжү§иЎҢе‘Ҫд»Өfab testе°ұеҸҜд»ҘзңӢеҲ°дёҖжқЎhello,worldиҫ“еҮәдәҶгҖӮ

0X02 жө…жҳҫзҡ„йҒ“зҗҶ

ж №жҚ®дёҠйқўз®ҖеҚ•зҡ„дҫӢеӯҗеҸҜд»ҘзңӢеҮәжқҘfabе‘Ҫд»Өжү§иЎҢзҡ„ж—¶еҖҷдјҡй»ҳи®ӨжүҫеҲ°еҪ“еүҚзӣ®еҪ•дёӢзҡ„fabfile.pyж–Үд»¶пјҢжүҫеҲ°еҗҺдјҡз”Ёfabе‘Ҫд»Өзҡ„еҸӮж•°еҺ»еҢ№й…Қfabfile.pyдёӯзҡ„еҮҪж•°еҗҚпјҢжү§иЎҢзӣёеә”зҡ„еҠҹиғҪгҖӮ

е®һйҷ…дёҠеҪ“еүҚзӣ®еҪ•еҸҜд»ҘжІЎжңүfabfile.pyпјҢеҰӮжһңеҪ“еүҚзӣ®еҪ•зҡ„дёҠзә§зӣ®еҪ•дёӯжңүfabfile.pyжҳҜдјҡйҮҮз”ЁдёҠзә§зӣ®еҪ•дёӯзҡ„fabfile.pyзҡ„гҖӮиҖҢдё”ж–Үд»¶еҗҚд№ҹдёҚдёҖе®ҡз”Ёfabfile.pyпјҢеҒҮи®ҫеҸ–дәҶдёҖдёӘеҗҚдёәasdf.pyзҡ„ж–Үд»¶пјҢйӮЈд№ҲеҸӘйңҖиҰҒжү§иЎҢfab -f asdf.pyе°ұеҸҜд»ҘйҮҮз”ЁиҝҷдёӘfabfileдәҶгҖӮ

Pythonдёӯзҡ„иҝӯд»ЈгҖҒз”ҹжҲҗе’Ңyieldе…ій”®еӯ—

еҸ‘иЎЁдәҺ 2017-11-12 жӣҙж–°дәҺ 2023-07-09

0X00 еҸҜиҝӯд»ЈеҜ№иұЎ

Pythonдёӯзҡ„еҲ—иЎЁпјҢе…ғз»„пјҢеӯ—е…ёпјҢж–Үд»¶йғҪжҳҜеҸҜиҝӯд»ЈеҜ№гҖӮеҸҜиҝӯд»ЈеҜ№иұЎз®ҖеҚ•ең°иҜҙе°ұжҳҜеҸҜд»Ҙз”Ёfor i in xxx:жқҘйҒҚеҺҶзҡ„еҜ№иұЎгҖӮ

дёҚиҝҮеҰӮжһңж•°жҚ®йҮҸйқһеёёйқһеёёеәһеӨ§зҡ„ж—¶еҖҷпјҢдјҡеҫҲеҪұе“ҚзЁӢеәҸзҡ„жҖ§иғҪгҖӮиҝҷз§Қж—¶еҖҷе°ұеҸҜд»ҘдҪҝз”Ёз”ҹжҲҗеҷЁжқҘи§ЈеҶіиҝҷдёӘй—®йўҳгҖӮ

0X01 з”ҹжҲҗеҷЁ

з”ҹжҲҗеҷЁзҡ„з”Ёжі•е’Ңжҷ®йҖҡзҡ„еҸҜиҝӯд»ЈеҜ№иұЎе·®дёҚеӨҡпјҢжңҖеӨ§зҡ„зү№зӮ№е°ұжҳҜпјҡвҖңз”Ёзҡ„ж—¶еҖҷжүҚеҺ»и®Ўз®—вҖқгҖӮиҝҷйҮҢеҶҷдёҖдёӘз®ҖеҚ•зҡ„дҫӢеӯҗпјҢжј”зӨәдёҖдёӢжғ…еҶөгҖӮз¬¬дёҖз§Қз”ҹжҲҗи¶…еӨ§еҲ—иЎЁзҡ„ж–№ејҸиҰҒйҖҗйЎ№и®Ўз®—е®ҢжүҚз®—еј„еҮәжқҘдәҶиҝҷдёӘ10000й•ҝзҡ„еҲ—иЎЁпјҢиҖҢеҗҺиҖ…жҳҜз”ҹжҲҗеҷЁпјҢеҸӘжҳҜеЈ°жҳҺдәҶжҖҺд№Ҳз®—пјҢе№¶жІЎжңүзңҹзҡ„еҺ»з®—пјҢжүҖд»ҘеңЁйҖҹеәҰдёҠжүҚжҳҜе®Ңе…ЁдёҚиғҪжҜ”зҡ„гҖӮ

0X00 жҖҺж ·жӯЈзЎ®дҪҝз”ЁеҲҶж”Ҝ

йҖҡеёёжғ…еҶөдёӢдёҖдёӘgitд»“еә“иҰҒдҝқжҢҒдёүдёӘеҸҠд»ҘдёҠзҡ„еҲҶж”ҜпјҢеҹәжң¬зҡ„еҲҶж”ҜжҳҺжҳҺеҰӮдёӢпјҡ

name function
master жӯЈеёёиҝҗиЎҢзҡ„зЁіе®ҡзүҲжң¬
develop жӯЈеёёиҝҗиЎҢзҡ„ејҖеҸ‘зүҲ

hotfix зҙ§жҖҘдҝ®еӨҚbugзҡ„еҲҶж”Ҝ

еҰӮжһңдҪ е·Із»ҸforkдәҶдёҖд»Ҫд»Јз ҒеҲ°иҮӘе·ұжң¬ең°пјҢеҪ“дҪ жғіж·»еҠ дёҖдёӘж–°еҠҹиғҪжҜ”еҰӮгҖҢз”ЁжҲ·з®ЎзҗҶгҖҚзҡ„ж—¶еҖҷпјҢе°ұеә”иҜҘе…ҲжҚўеҲ°developеҲҶж”ҜпјҢз„¶еҗҺз”ұиҝҷдёӘеҲҶж”ҜеҲӣе»әдёҖдёӘж–°зҡ„еҗҚдёәfeture_add_usermanagerзҡ„еҲҶж”ҜгҖӮеңЁж–°еҲҶж”ҜйҮҢзј–еҶҷд»Јз ҒеҗҺе°Ҷд»Јз ҒжҸҗдәӨдёҖдёӘPullRequestеҲ°иҮӘе·ұзҡ„developеҲҶж”ҜпјҢеҗҲе№¶иө·жқҘеҗҺеҶҚжҸҗдәӨдёҖдёӘPullRequestеҲ°еӣўйҳҹзҡ„д»“еә“дёӯпјҢзӯүеҫ…еӣўйҳҹе…¶д»–жҲҗе‘ҳreviewеҗҺе°ұеҸҜд»ҘжӯЈејҸе°Ҷд»Јз ҒеҗҲе№¶еҲ°еӣўйҳҹзҡ„developдёӯдәҶгҖӮзӯүдёӢдёҖж¬ЎеҸ‘еёғж–°зүҲжң¬зҡ„ж—¶еҖҷе°ұеҸҜд»Ҙе°Ҷеӣўйҳҹзҡ„developеҲҶж”ҜеҗҲе№¶еҲ°еӣўйҳҹзҡ„masterеҲҶж”ҜдёӯдәҶдәҶгҖӮ

еҰӮжһңдёӯйҖ”йЎ№зӣ®еҮәзҺ°дәҶдёҘйҮҚbug(дёҚиғҪзҷ»еҪ•)йңҖиҰҒеҚідҪҝдҝ®ж”№дёҠзәҝпјҢйӮЈе°ұд»ҺиҮӘе·ұзҡ„masterеҲҶж”ҜдёҠж–°е»әдёҖдёӘеҗҚдёәhotfix_cantloginзҡ„еҲҶж”ҜпјҢдҝ®ж”№е®ҢеҗҺзӣҙжҺҘжҸҗдәӨPullRequestеҲ°еӣўйҳҹзҡ„masterеҲҶж”ҜпјҢreviewдё”еҗҲе№¶еҗҺе°ұеҸҜд»Ҙе°ҶиҜҘеҲҶж”ҜеҲ йҷӨдәҶгҖӮ
