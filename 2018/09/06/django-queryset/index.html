Djangoдёӯзҡ„дёҖдәӣйқһе…Ҙй—Ёзә§з”Ёжі•

еҸ‘иЎЁдәҺ 2018-09-06 жӣҙж–°дәҺ 2023-07-09

дёәд»Җд№ҲиҝҷйҮҢиҜҙжҳҜвҖқйқһе…Ҙй—Ёзә§вҖқз”Ёжі•е‘ўпјҢеӣ дёәжҲ‘дёӘдәәи§үеҫ—иҝҷжҳҜжҲ‘жҺҘи§ҰDjangoд№ӢеҗҺдёҖж®өж—¶й—ҙжүҚејҖе§ӢдәҶи§Јзҡ„з”Ёжі•пјҢдҪҶжҳҜиҜҙжҳҜй«ҳзә§з”Ёжі•еҸҲеӨӘеӨёеј дәҶпјҢжүҖд»Ҙз”ЁдәҶиҝҷд№ҲдёҖдёӘиҜЎејӮзҡ„вҖқйқһе…Ҙй—Ёзә§вҖңзҡ„е®ҡдҪҚгҖӮ

дёӢйқўзҡ„зӨәдҫӢдёӯдҪҝз”ЁдёӢйқўзҡ„modelпјҢз®ҖеҚ•жҸҸиҝ°дёҖдёӢе№¶йқһзңҹе®һд»Јз Ғ

0X00 дҪҝз”ЁAvg()/Sum()/Count()/Max()/Min()

иҝҷдәӣж–№жі•зҡ„з”Ёжі•еҫҲз®ҖеҚ•пјҢйЎҫеҗҚжҖқд№үгҖӮдёҚиҝҮйңҖиҰҒй…ҚеҗҲдёӢйқўд»Ӣз»Қзҡ„annotate()жҲ–aggregate()дҪҝз”ЁгҖӮ

from django.db.models import Avg, Sum, Count, Max, Min

name function
Avg жұӮе№іеқҮж•°
Sum жұӮе’Ң
Count и®Ўж•°
Max жұӮжңҖеӨ§
Min жұӮжңҖе°Ҹ

0X01 дҪҝз”Ёannotate()

жңҖеҹәзЎҖзҡ„жҹҘиҜўе°ұжҳҜд»ҺдёҖеј иЎЁдёӯжҹҘиҜўз¬ҰеҗҲжҹҗжқЎд»¶зҡ„еӯ—ж®өпјҢиҖҢдҪҝз”Ёannotate()еҸҜд»Ҙеҫ—еҲ°дёҖдәӣжҲ‘д»¬жүӢеҠЁи®Ўз®—еҫ—еҲ°зҡ„еҖјпјҢе№¶е°Ҷе…¶дҪңдёәQuerysetдёӯItemзҡ„дёҖдёӘеұһжҖ§жқҘи°ғз”ЁгҖӮ

еҰӮжһңжҲ‘жғіиҰҒжҹҘиҜўжҜҸдёӘдәә(Staff)жүӢдёӢжңүеӨҡе°‘дёӘи®ўеҚ•(Order)пјҢйӮЈд№ҲиҜҘжҖҺд№ҲжҹҘе‘ўпјҢдҪҝз”ЁеҲқзә§зҡ„з”Ёжі•еҸҜд»ҘеҶҷеҮәзұ»дјјдёӢйқўзҡ„д»Јз ҒгҖӮдҪҶжҳҜжңүдёҖдёӘжҜ”иҫғдёҘиӮғзҡ„й—®йўҳпјҡдјҡдә§з”ҹз”ЁжҲ·ж•°йҮҸ + 1ж¬Ўзҡ„жҹҘиҜўгҖӮиҝҷйҮҢеҸӘжңүе°‘ж•°з”ЁжҲ·й—®йўҳдёҚеӨ§пјҢеҰӮжһңжңүдёҠеҚғз”ҡиҮідёҠдёҮдёӘз”ЁжҲ·пјҢйӮЈд№Ҳе°ұдјҡдә§з”ҹеҮ еҚғеҮ дёҮж¬ЎжҹҘиҜўпјҢйӮЈеҜ№ж•°жҚ®еә“зҡ„еҺӢеҠӣжҳҜеҫҲжҒҗжҖ–зҡ„гҖӮ

дҪҝз”Ёannotate()ж–№жі•е°ұеҸҜд»Ҙжңүж•Ҳи§ЈеҶіиҝҷдёӘй—®йўҳ

иҝҷйҮҢзҡ„staff_order_countеӯ—ж®өжҳҜиЎЁдёӯе№¶дёҚеӯҳеңЁзҡ„пјҢжҳҜйҖҡиҝҮannotate()ж–№жі•дёҙж—¶еӯҳеӮЁзҡ„дёҖдёӘеӯ—ж®өгҖӮеҗҢж ·зҡ„пјҢеҶҚдёҖдёӘannotate()ж–№жі•дёӯеҸҜд»ҘеҠ е…ҘеӨҡдёӘеҸӮж•°пјҢдҪҝз”ЁеҗҢж ·зҡ„ж–№жі•еҺ»з»ҹи®Ўе’ҢиҺ·еҸ–ж•°жҚ®еҚіеҸҜгҖӮ

еңЁannotate()дёӯдҪҝз”ЁCount()дёҖе®ҡиҰҒжҳҜжңүеӨ–й”®е…іиҒ”жүҚиЎҢгҖӮдҫӢеҰӮжң¬дҫӢдёӯпјҢOrderиЎЁдёӯжңүдёҖдёӘеӨ–й”®еӯ—ж®өstaffдёҺиЎЁStaffе…іиҒ”иө·жқҘдәҶпјҢйӮЈд№Ҳе°ұеҸҜд»ҘеңЁDjangoдёӯйҖҡиҝҮStaff.order_setжқҘиҺ·еҸ–е…іиҒ”еҲ°Staffзҡ„OrderпјҢжүҖд»Ҙд№ҹе°ұеҸҜд»ҘдҪҝз”ЁCount()ж–№жі•жқҘиҝӣиЎҢз»ҹи®ЎдәҶгҖӮ

з”ҹжҲҗжҹҘиҜўзҡ„SQLиҜӯеҸҘд№ҹжү“еҮәжқҘж–№дҫҝзҗҶи§ЈгҖӮ

0X02 дҪҝз”Ёaggregate()

дҪҝз”Ёaggregate()еҸҜд»ҘдҪҝеҫ—жҹҘиҜўзҡ„иҝ”еӣһеҖјз”ұдёҖдёӘQuerysetеҸҳжҲҗдёҖдёӘdictпјҢжҜҸдёӘkeyе’ҢеҜ№еә”зҡ„valueз”ұиҮӘе·ұи®Ўз®—еҫ—еҲ°гҖӮ

еҰӮжһңжҲ‘йңҖиҰҒи®Ўз®—еҮәжүҖжңүдәәдёӯе№ҙйҫ„жңҖеӨ§гҖҒжңҖе°ҸгҖҒе№іеқҮеҖјиҜҘжҖҺд№ҲеҠһпјҹеҲқзә§з”Ёжі•еҸҜиғҪйңҖиҰҒе…Ҳз”ЁдёҖдёӘжҹҘиҜўеҫ—еҮәжүҖжңүдәәзҡ„е№ҙйҫ„пјҢз„¶еҗҺеҶҚеҚ•зӢ¬еҺ»и®Ўз®—жңҖеӨ§жңҖе°Ҹе№іеқҮеҖјгҖӮеҶҷеҮәзұ»дјјеҰӮдёӢд»Јз ҒпјҢиҷҪз„¶зӣ®еүҚй—®йўҳдёҚеӨ§пјҢдёҚиҝҮеҪ“йҖ»иҫ‘еӨҚжқӮиө·жқҘд№ӢеҗҺе°ұдјҡйҡҫд»ҘзҗҶи§Је№¶дё”д»Јз ҒйҮҸиҫғеӨ§гҖӮ

дҪҶжҳҜеҰӮжһңдҪҝз”Ёaggregate()ж–№жі•еҶҷеҮәдёҚд»…йҖ»иҫ‘жё…жҷ°дёҚжҳ“еҮәй”ҷпјҢиҖҢдё”д»Јз ҒйҮҸе°‘дәҶеҫҲеӨҡпјҢжӣҙз®ҖеҚ•жҳ“иҜ»гҖӮ

иҝҷж®өд»Јз Ғе°ұдјҡзӣҙжҺҘиҫ“еҮәеҰӮдёӢdictпјҢйңҖиҰҒзҡ„ж•°жҚ®зӣҙжҺҘеҸ–еҚіеҸҜгҖӮ

0X03 дҪҝз”ЁCase/When

Djangoдёӯзҡ„Case()/When()жҳҜйқһеёёе®һз”ЁдёҖеҜ№ж–№жі•пјҢжҒ°еҪ“дҪҝз”ЁеҸҜд»ҘеӨ§е№…еәҰеҮҸе°Ҹз»ҹи®ЎеҠҹиғҪзҡ„д»Јз ҒйҮҸгҖҒйҖ»иҫ‘еӨҚжқӮеәҰзӯүгҖӮ

еҒҮи®ҫжңүеҰӮдёӢйңҖжұӮвҖқе№ҙйҫ„е°ҸдәҺ18зҡ„дёәжңӘжҲҗе№ҙ(1)пјҢе№ҙйҫ„еңЁ1930д№Ӣй—ҙзҡ„дёәйқ’е№ҙ(2)пјҢе№ҙйҫ„еңЁ3160зҡ„дёәдёӯе№ҙ(3)пјҢе…¶д»–дёәиҖҒе№ҙ(0)вҖңпјҢйӮЈд№ҲдҪҝз”ЁCase/Whenж–№жі•еҶҚй…ҚеҗҲannotate()ж–№жі•е°ұеҸҜд»Ҙдјҳйӣ…еҫ—е®һзҺ°еҠҹиғҪгҖӮ

дёҠйқўжҳҜи®Ўз®—дёҖдёӘQueyrSetдёӯжҜҸдёҖдёӘitemзҡ„жғ…еҶөпјҢиҝҳжңүдёҖз§Қжғ…еҶөжҳҜз»ҹи®ЎдёҖдёӘmodelдёӯжүҖжңүж•°жҚ®пјҢдҫӢеҰӮиҝҷдёӘйңҖжұӮпјҡвҖқз»ҹи®ЎжүҖжңүOrderдёӯпјҢеҚ•д»·жңҖй«ҳгҖҒжңҖдҪҺе’Ңе№іеқҮеҖјвҖңгҖӮдҪҝз”ЁCase()/When()д№ҹеҸҜд»Ҙе®ҢжҲҗд»»еҠЎпјҢе№¶дё”жҜ”еҲқзә§з”Ёжі•жӣҙеҘҪдёҖдәӣгҖӮ

еҶ…е®№ж•ҙзҗҶзҡ„жңүзӮ№е·®пјҢеҗ„дҪҚеҸ‘зҺ°дәҶд»Җд№Ҳз–ҸжјҸе’Ңй”ҷиҜҜиҜ·еҸҠж—¶иҒ”зі»жҲ‘пјҢйҳІжӯўиҜҜеҜјеҲ«дәәгҖӮеҰӮжһңж–Үз« еҜ№еӨ§е®¶еёҰжқҘдәҶеё®еҠ©пјҢйӮЈжҲ‘иҝҳжҳҜеҫҲејҖеҝғзҡ„ еҳҝеҳҝ
